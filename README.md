# BESSTIE-NLP-Classifier 🚀

A multi-variety English sentiment and sarcasm classifier built on the **BESSTIE** benchmark. 

This project addresses linguistic bias in sentiment analysis by evaluating and classifying text across three varieties of English: **Australian**, **British**, and **Indian**.

## ✨ Features
- **Multi-Variety Support**: Fine-tuned classification for diverse English dialects.
- **RoBERTa-based Classification**: Leverages state-of-the-art transformer models.
- **Classical Baselines**: Includes Logistic Regression baselines.
- **Interactive UI**: Built with Gradio for real-time prediction.

## 🛠️ Tech Stack
- **Languages**: Python
- **Frameworks**: Transformers, PyTorch, Gradio
- **Models**: RoBERTa, Logistic Regression

## 🚀 Getting Started

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/DeepSoul-173/BESSTIE-NLP-Classifier.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the App
To launch the Gradio interface:
```bash
python app.py
```
